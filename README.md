# Task Manager API

Task management REST API built with Spring Boot and PostgreSQL.

## Technologies

- Java 21
- Spring Boot
- PostgreSQL
- Maven
- Swagger

## Features

- Create tasks
- Update tasks
- Delete tasks
- Get task by ID
- List all tasks
